JQuery的选择器

基本选择器 
匹配所有元素    $('*')
获取ID        $('#id')
获取元素       $('div')
获取class      $('.class')
  ★注意：相同元素下、可指定选择起带clss
  同元素中带指定class  $('ele.class')
多元素获取      $('ele,ele')

事例
<div id='a'>div/ID:a</div>
<h3>H3元素</h5>
<h5 class="h5">H5元素/class：h5</h5>
<h5><span>H5 下子元素 span</span></h5>
<p>P元素</p>
<h2>H2元素</h2>
<h5 class="newh5">H5/class:new</h5>

<script>
	$(function(){
		$('*').css('margin','20px');//所有元素
		$('#a').css('color','red');
		$('h3').css('color','lightyellow');
		$('.h5').css('color','blue');
		$('h5 span').css('color','green');
		$('p,h2').css('color','skyblue');//多个元素以，分隔
		$('h5.newh5').css('color','red');//ele.class
	})
</script>
属性选择器 
[attribute] 	
[attribute=value]
[attribute!=value]★注意：谨慎使用

attribute value属性值。引号在大多数情况下是可选的。但在遇到诸如属性值
包含"]"时，用以避免冲突。

<div a='b'>DIV元素/a:b</div>
<div a='c'>DIV元素/a:c</div>
<div b='a'>DIV元素/b:a</div>
<h1 c='a'>h1元素/c:a</h1>
<h1 c='b'>h1元素/c:b</h1>
<h1 c='c'>h1元素/c:c</h1>
<h1 d='a'>h1元素/c:c</h1>
<h1>h1元素/不含自定义元素</h1>
<script>
	$(function(){
		$('[a]').css('color','red');
		$('[a=c]').css('color','darkgoldenrod');
		$('[b=a]').css('color','blue');
		$('h1[c!="a"]').css('color','red');
	  //注意使用这个！、会把所有的元素不包含都定属性、使用加上元素
	})
</script>

[attribute^=value] ^=value 以value开始的元素<br />
[attribute$=value] $=value 以value结尾的元素<br />
[attribute*=value] *=value 包含含有value的元素<br />
[attrSel1][attrSel2][attrSelN] 多个自定义元素
层级选择器 
ancestor descendant 元素下所有相同属性
parent > child  元素下子辈属性、不包括子辈以下
prev + next 相邻的两个元素、对第2个元素操作
prev ~ siblings 同辈的 siblings 元素

子元素 
ancestor descendant 元素下所有相同属性
parent > child  元素下子辈属性、不包括子辈以下
                	
必须第1个元素为 son元素
parent son:first-child //匹配 第1个parent元素 里面的第1个 son元素 
parent son:last-child //匹配 最后1个parent元素 里面的最后1个 son元素
若多个和 parent相同的元素也只能匹配 第1个/最后1个 son 元素

parent son:first-of-type 1.9+ //匹配 parent元素 里的 第1个son 元素
parent son:last-of-type 1.9+ //匹配 parent元素 里的 最后1个son 元素
多个和 parent相同的元素匹配多个里的 第1个/最后1个 son 元素

nth数组由1开始计算
parent son:nth-child
//可对多个想通parent操作、指定 son元素 要在parent相应位置
parent son:nth-child(even)//对 parent元素 的 son元素的偶数
//若son是梅花间竹、则为全部
parent son:nth-child(odd)//对 parent元素 的 son元素的基数
//若son是梅花间竹、则为没效果

parent son:nth-of-type() 1.9+ //对相同的parent元素内的son元素操作

parent son:nth-last-child() 1.9+//对相同的parent元素内的son元素操作
//若son是梅花间竹、odd没效果、even可以

parent son:nth-last-of-type() 1.9+//对相同的parent元素内的son元素操作
////不需要匹配相应的son

parent son:only-child//该parent只有1个son元素操作
//若parent元素下有除son元素其他的元素不能生效

parent son:only-of-type 1.9+
//若parent元素下有除son元素能生效
