Boolean类型

关于Boolean 
boolean类型型是JavaScript中的一种原始类型，它只有两种值：true和false。
定义Boolean 
定义对象为Boolean类型
var a = true;
var b = false;
对象转换Boolean 
转换Boolean值、Boolean(obj）
var a = "";
console.log(Boolean(a)); //false

//1.特殊值undefined和null变成false.
//2.数字0和NaN变成false.
//3.空字符串变成false.
//4.其他所有值都变成true.

//事例在if上、因为else里定义没有var 声明、a为全局对局
var a = new Object();
if (a.a) {
  console.log(a.a);
} else {
  a.a = "just a";
  console.log(a.a);  //just a
}
引用Boolean类型 
var a = new Boolean(true);
var b = new Boolean(false);

console.log(a && false);//flase
console.log(false && a);//flase
console.log(true && b);//Boolean {[[PrimitiveValue]]: false}

console.log(b && true);//true
//当执行b && true时，b被强制转换类型为Obj，
//强制转换为true。这边用来计算的是b对象，而不是它的值false

JavaScript一种直译式脚本语言，是一种动态类型、弱类型、基于原型的语言，内置支持类型。它的解释器被称为JavaScript引擎，为浏览器的一部分，广泛用于客户端的脚本语言，最早是在HTML（标准通用标记语言下的一个应用）网页上使用，用来给HTML网页增加动态功能。 为了取得技术优势，微软推出了JScript，CEnvi推出ScriptEase，与JavaScript同样可在浏览器上运行。为了统一规格，因为JavaScript兼容于ECMA标准，因此也称为ECMAScript。 在1995年时，由Netscape公司的Brendan Eich，在网景导航者浏览器上首次设计实现而成。因为Netscape与Sun合作，Netscape管理层希望它外观看起来像Java，因此取名为JavaScript。但实际上它的语法风格与Self及Scheme较为接近。
