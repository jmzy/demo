函数的返回

1、函数默认返回值为undefiend

2、可指定return的位置，超出函数体
   即：return document;//该document为script主体

3、函数体里返回document，且 document 只含有1个函数并同一函数名时
   该函数在进行是，且匿名函数值为函数名
   结果为；先执行函数，click后再次执行
   即：function a(){alert(1);return document;};
   a().onclick=a;
   //alert 1;[再次click]alert 1;

4、对象事件等于值为函数名，但没有进行时、
   函数体内的返回值为function(){}、不执行
   对象事件等于值为函数名，
   ★进行时、函数体内的返回值为function(){}、不执行
   ★注意：是事件后等于匿名函数中的函数名需要在进行时
    obj.onclick=a();//★a、不执行、a()执行
      function a(){
      return function b(){
      alert(1);};
   };

5、return作用，在★实名函数体内阻断后续代码、
   进行断点排除错误、后续代码亦出错不影响
   ★注意：若return为其他内容后续代码亦出错受影响

6、函数体内可以存在多个return

JavaScript一种直译式脚本语言，是一种动态类型、弱类型、基于原型的语言，内置支持类型。它的解释器被称为JavaScript引擎，为浏览器的一部分，广泛用于客户端的脚本语言，最早是在HTML（标准通用标记语言下的一个应用）网页上使用，用来给HTML网页增加动态功能。 为了取得技术优势，微软推出了JScript，CEnvi推出ScriptEase，与JavaScript同样可在浏览器上运行。为了统一规格，因为JavaScript兼容于ECMA标准，因此也称为ECMAScript。 在1995年时，由Netscape公司的Brendan Eich，在网景导航者浏览器上首次设计实现而成。因为Netscape与Sun合作，Netscape管理层希望它外观看起来像Java，因此取名为JavaScript。但实际上它的语法风格与Self及Scheme较为接近。
