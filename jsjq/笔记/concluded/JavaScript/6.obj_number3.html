对象中的数值获取方法

仅对数值、若obj不确定是否数值可对其先返回数值

1、parseInt(string,radix)可以获得负数值

    string必需。要被解析的字符串。
    例：a = "123adf";
        console.log(parseInt(a) + "---a");//123

        console.log(parseInt("dda123"));//NaN第1位必须是数字、否则NaN
        console.log(parseInt("0.123"));//0取整数、小数点后不取
        console.log(parseInt("-1"));//0-1
        console.log(parseInt(" -1"));//0忽略空格
        console.log(parseInt(" -1 231"));//1
        忽略空格、若数值前后均有空格、取第1次的空格值
        console.log(parseInt("3+2a"));//3运算符不作运算、也不做连接
        console.log(parseInt("3-2-a"));//3-、/、*、%...都不行
        console.log(parseInt("2-1-2"));//2全数字也不行

    radix可选
    ★这里不懂
    ★parseInt(obj,x)
    可以传两个值、obj若是字符需""、x为进制

2、parseFloat(string)可以获得负数并小数点

    string必需。要被解析的字符串
    例：a = "123adf";
        a = "123.45adf";parseFloat(a);//123.45

        console.log(parseFloat("a123"));//NaN
        console.log(parseFloat("123.45adf"));//123.45
        console.log(parseFloat("123.45.67"));//123.45
        可保留小数点后的位数、紧保留1次
        console.log(parseFloat(" -12.34.5"));//-12.34
        console.log(parseFloat(" -12.1234567890123456789"));
        //-12.1234567890123456★仅可以保留小数点后16位小数、

3、number(obj)
    a ="100";console.log(number(a));/100
    number()转换数值、仅对obj为纯数字、若obj喊符号、字母则为NAN
    boolean类型特殊转换
    a = trun;//1; a = flase;//0 a = null;//0

    例：console.log(Number(new Boolean(true)));//1
        console.log(Number(new Boolean(false);));//0
        console.log(Number(null));//0
        console.log(Number("-999 888"));//-999空格前正确数值
        console.log(Number(" -999 888"));//-999
        可保留小数点后的位数、紧保留1次
        console.log(Number("1+1"));//NaN
        运算符不作运算、也不做连接
        console.log(Number("1-1"));//NaN
        console.log(Number("a1"));//NaN
        字母在前后都是NaN
        console.log(Number("1a"));//NaN

4、★强制转换
    a = "134345";console.log(a*1);//134345
    ★含符号、字母等就不能转换了
