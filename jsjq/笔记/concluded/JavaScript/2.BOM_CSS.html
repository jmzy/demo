JavaScript HTML DOM - 改变 CSS样式 方法一

改变 HTML CSS样式、方法一
DOM.style.property=new style

★// 查看样式所支持的属性和方法。 console.log(dom.style)

DOM.style.property=new style
document.getElementById(id).style.property = new style
例：
<div id="div"></div>
<script>
var dom = document.getElementById("div");
dom.style.width = "100px";
dom.style.border = "10px solid #000";
dom.style.backgroundColor = "pink";
dom.style.marginLeft = "10px";

/*
出现两个单词，或多个的单词的， 需要第二个单词往后，首字母大写
border-width: borderWidth
border-radius: borderRadius
*/

</script>

JavaScript HTML DOM - 改变 CSS样式 方法二

改变 HTML CSS样式、方法二

DOM.style.cssText = "new style" 单个样式修改

DOM.style.cssText = "new style1;new style2;"
  设置多个属性、会覆盖掉之前的所有内联样式

DOM.style.cssText += "new style"（original style + new style）
  保留原来样式并添加新样式

例：

<div id="div"></div>

<script>
var dom = document.getElementById("div");

//单个样式修改
dom.style.cssText = "width:100px";
dom.style.cssText = "height:100px";

//多个样式、以，逗号为分隔符
dom.style.cssText = "width:100px,height:100px";
★// cssText 可以同时设置多个属性、会覆盖掉之前的所有内联样式.

★// 如果需要保留之前的样式，需要使用 +=
★dom.style.cssText += "width:10px;heihgt:20px;backgroound:skyblue";
</script>

JavaScript HTML DOM - 改变 CSS样式 方法三

改变 HTML CSS样式、方法三

★命名新的class并定义新的className

1、添加新的className
<style>
.two{width:100px}
</style>

<div id="div"></div>

<script>
var obj = document.getElementById("div");
obj.className="two";
</script>

★//多个样式、直接在定义新的className添加
<style>.two{width:100px;height:100px}</style>
obj.className="two";
</script>

★//若对象原来已有className需在引号下添加空格（" two")
obj.className=" two";

★//补充、删除className、
obj.className=" ";

★若需要保留原有style需把原来的className写上
obj.className="one";

JavaScript一种直译式脚本语言，是一种动态类型、弱类型、基于原型的语言，内置支持类型。它的解释器被称为JavaScript引擎，为浏览器的一部分，广泛用于客户端的脚本语言，最早是在HTML（标准通用标记语言下的一个应用）网页上使用，用来给HTML网页增加动态功能。 为了取得技术优势，微软推出了JScript，CEnvi推出ScriptEase，与JavaScript同样可在浏览器上运行。为了统一规格，因为JavaScript兼容于ECMA标准，因此也称为ECMAScript。 在1995年时，由Netscape公司的Brendan Eich，在网景导航者浏览器上首次设计实现而成。因为Netscape与Sun合作，Netscape管理层希望它外观看起来像Java，因此取名为JavaScript。但实际上它的语法风格与Self及Scheme较为接近。
