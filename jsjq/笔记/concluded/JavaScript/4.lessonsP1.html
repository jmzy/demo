注意点
   一、变量未赋值、为undefined//默认返回

   二、运用数据之前与之后

      实名函数调用不受前后调用限制、
         ★非进行时候，function(){} 函数体
         ★进行时候，undefiend

      匿名函数调用★受前后调用限制、
         ★非进行时候、前undefiend、后function(){} 函数体
         ★★进行时候，前报错 、后undefiend

         非function、定义变量赋值为
         之前undefined；之后为赋值值
         例：alert(a);var a=1;//alert undefined
         var a=str;alert(a);//alert 1

      调用实名函数进行时，非声明function、不存在前后调用不同
      例：
      console.log(a());//alert1;★function默认返回undefined
      function a(){alert(1)};
      console.log(a());//alert1;★function默认返回undefined

   三、外部变量名与声明实名函数体内变量名想同、函数体内外变量名均赋值、
      调用时、函数是否调用都不影响变量名在函数体外的复制、
      例：
         a=1;
         b();
         function b(){a=2};
         alert(a);//1
         ★除非函数体来return 该变量名
         例：a=1;
         b();
         function b(){return a=2};
         alert(a);//1

JavaScript一种直译式脚本语言，是一种动态类型、弱类型、基于原型的语言，内置支持类型。它的解释器被称为JavaScript引擎，为浏览器的一部分，广泛用于客户端的脚本语言，最早是在HTML（标准通用标记语言下的一个应用）网页上使用，用来给HTML网页增加动态功能。 为了取得技术优势，微软推出了JScript，CEnvi推出ScriptEase，与JavaScript同样可在浏览器上运行。为了统一规格，因为JavaScript兼容于ECMA标准，因此也称为ECMAScript。 在1995年时，由Netscape公司的Brendan Eich，在网景导航者浏览器上首次设计实现而成。因为Netscape与Sun合作，Netscape管理层希望它外观看起来像Java，因此取名为JavaScript。但实际上它的语法风格与Self及Scheme较为接近。
