PHP 流程控制

什么是流程控制 
PHP属于动态语言、很多情况下判断后需要执行分支(场景：登录)
流程控制是在特定的条件下执行流程分支
判断条件的语法有
if、if...else、switch、
循环语句
while、do...while
for循环
if 和 if..else 和 if..elseif...else 语句 
if(表达式){//当表达式为 true 的情况下 执行代码段
	代码段
}

if(表达式){//当表达式为 true 的情况下 执行代码段
	代码段
}else{//当表达式为 false 的情况下 执行代码段
	代码段
}

if(表达式1){//当满足表达式1的情况下 执行代码段
	代码段
}elseif(表达式2){//当满足表达式2的情况下 执行代码段
	代码段
}else{//当不能满足表达式1和2的情况下 执行代码段
	代码段
}

注意if语句中、1个条件成功、其他分支不执行、if的表达式只能为bool类型
switch语句 
switch(表达式){
	case 值1：//表达式值为1执行代码段
	... 代码段;
	break;
	case 值2：//表达式值为2执行代码段
	... 代码段;
	break;
	case 值3：//表达式值为3执行代码段
	... 代码段;
	break;
	default;//终止switch执行
}
注意 switch语句中、表达式一般为变量
  如果不架break;后面的case都会执行
  即判断完case1再次判断case2
  场景：$a=10、case1 $a > 0; case2 $a >5;
while语句 
★注意1定要先设好php ini、不然会死机
max_execution_time = 30单位是s
通常表达式里会设置条件、例如小于大于
while(表达式){//当表达式为true 执行代码段
	代码段
}

事例 --- 99乘法表
<?php
  $i = 1;
  while($i <= 9){
    $j = 1;
    while($j <= $i){
      echo $i.'*'.$j.'='.$i*$j."   ";
        //\t水平制表(HT) （跳到下一个TAB位置）
      j++;//先执行后++
    };
    echo '<br/>';
    $i++;//先执行后++
  }
?>

跟随 INT 意义为退出几层的循环
$i = 1;
$j = 1;
while($i++ < 10){
	echo '$i = '.$i;
	while($j++ <5){
		if($j == 2){
			break 2;
		}
		echo '$nbsp;  $j = '.$j;
	}
	echo '<br/>';
}
continue 直跳运行到循环体的结尾
continue 跟随 INT 意义为跳出int层的循环并返回到循环头部 
$i=0;
$j=0;
while($i++ < 5){
	if(++$j == 2){
		continue;
	}
	echo '$i='.$i.'<br/>';
}
//$i = 1; $j = 1;  $i = 3; $j = 3;  $i = 4; $j = 4;  $i = 5; $j = 5;  

跟随 INT 意义为跳出int层的循环并返回到循环头部
while(++$a < 3){
	while(++$b < 4){
		if($b == 2){
			echo '  $b:'.$b;
		}
	}
	$b = 0;
	echo '$a:'.$a;
	echo '<br/>';
}

for($i=1;$i<5;$i++){
	if($i == 2){
		continue;
	}
	echo $i.'<br/>';
}
exit die
