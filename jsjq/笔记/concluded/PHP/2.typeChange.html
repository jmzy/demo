PHP 类型转换

自由转换 
变量值的改变
$a = '100';
$a = 100;
转换 字符串（string） 
整型（int） 转换 字符串（string）-----(string)$obj ;
$a = 100;
$b = (string)$a;
var_dump($a);//int 100
var_dump($b);//string '100' (length=3)
settype($a,'string');
var_dump($a);//string '100' (length=3)

浮点（float）转换字符串（string） -----(string)$obj
$f1 = 2017.5;
$f2 = (string)$f1;
var_dump($f2);//string '2017.5' (length=6)
settype($f1,'string');
var_dump($f1);//string '2017.5' (length=6)
转换 整型（int） 
字符串（string） 转换 整型（int）-----(int)$obj
$c = '100abc';
$d = (int)$c;
$e = '100abc1111';
$f = (int)$e;
$g = 'aaa100abc1111';
$h = (int)$g;
var_dump($c);//string '100abc' (length=6)
var_dump($d);//int 100
var_dump($e);//string '100abc1111' (length=10)
var_dump($f);//int 100 只能截取最前面的数字
var_dump($g);//string 'aaa100abc1111' (length=13)
var_dump($h);//int 0字符串最前非数字截取后为int类型 0

浮点（float） 转换 整型（int）-----(int)$obj
$a1 = '2017.5.21 礼拜天';
$c2 = (float)$a1;
var_dump($c2);//float 2017.5只能转第1次小数点的内容
$e1 = 'aaa2017.5.21 礼拜天';
$e2 = (float)$e1;
var_dump($e2);//float 0字符串最前非浮点数借钱为float类型0
转换 浮点（float） 
字符串（string） 转换浮点（float）-----(float)$obj
$a1 = '2017.5.21 礼拜天';
$a2 = (int)$a1;
$b1 = '2017.5.21';
$b2 = (int)$b1;
var_dump($a1);//string '2017.5.21 礼拜天' (length=19)
var_dump($a2);//int 2017
var_dump($b1);//string '2017.5.21' (length=9)
var_dump($b2);//int 2017
$c2 = (float)$a1;
var_dump($c2);//float 2017.5
settype() 设置变量类型转换-- 作用于原变量 
$a = 100;
settype($a,'string');
var_dump($a);//string '100' (length=3)
具体转换函数 intval() folatval() strval() 
只针对变量类型 str int float string bool
intval() 转换为整形
folatval() 转换为浮点型
strval() 转换为字符串

$ta1 = '100';
$ta2 = intval($ta1);
$ta3 = floatval($ta1);
$ta4 = strval($ta1);
var_dump($ta2);//int 100
var_dump($ta3);//float 100
var_dump($ta4);//string '100' (length=3)

JavaScript一种直译式脚本语言，是一种动态类型、弱类型、基于原型的语言，内置支持类型。它的解释器被称为JavaScript引擎，为浏览器的一部分，广泛用于客户端的脚本语言，最早是在HTML（标准通用标记语言下的一个应用）网页上使用，用来给HTML网页增加动态功能。 为了取得技术优势，微软推出了JScript，CEnvi推出ScriptEase，与JavaScript同样可在浏览器上运行。为了统一规格，因为JavaScript兼容于ECMA标准，因此也称为ECMAScript。 在1995年时，由Netscape公司的Brendan Eich，在网景导航者浏览器上首次设计实现而成。因为Netscape与Sun合作，Netscape管理层希望它外观看起来像Java，因此取名为JavaScript。但实际上它的语法风格与Self及Scheme较为接近。
