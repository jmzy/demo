PHP 数组

什么是数组 
在1个变量中、存储1个或多个值、每1个元素都有访问ID(索引值)
数组中的值可以存放任意类型
数组中的值为true、遍历为1、false、0、
数组中的值为null不显示、但也占着位置
定义数组 和 提取数组值/遍历数组 、获得数组长度(数组里元素的数量) 
定义数组
1.直接赋值创建
 $arr1 = array('a','b','c');
 echo '26个字母、第1个是'.$arr1[0].'第2个是'.$arr1[1].'第3个是'.$arr1[2];

2.$arr2[0] = 'a';
 echo $arr2[0];
 var_dump($arr2);

遍历数组 
for($i=0;$i<3;$i++){
	echo $i.'--'.$arr1[$i].'<br/>';
};

获得数组长度
echo count($arr1).'<br/>';//3

3.指定下标
 $arr3['a'] = 'a';
 $arr3['b'] = 'b';
 
 遍历/获得的跟js json 差不多
 $arr3['a'] = 'a';
 $arr3['b'] = 'b';
 foreach($arr3 as $key=>$val){
   echo $key.'='.$val.'<br/>';
 };
 ★注意foreach遍历不需要从0开始
 
4.数组是可以动态增长、键值可以跳跃、
数组求总和 和 数组平均值 
$arr4 = array(1,2,3,4,5,2,34,50);
$allNum = null;
for($i=0;$i<count($arr4);$i++){
	$allNum += $arr4[$i];
};
echo $allNum.'<br/>';//101
echo $allNum/count($arr4).'<br/>';//12.625
echo count($arr4).'<br/>';//8
数组下标[键值] 
1.指定下标起始值
$arr5 = array(5=>'e','f','g');
echo count($arr5).'<br/>';
echo $arr5[5].'<br/>';//3
//	echo $arr5[3].'<br/>';// Undefined 
echo $arr5[6].'<br/>';//f
echo $arr5[7].'<br/>';//g

2.bool和null('' "")作为下标
$arr6[true]= 1;
$arr6[false]= 0;
$arr6[null]= 'a';
echo count($arr6).'<br/>';//3
echo $arr6[0].'<br/>';//0
echo $arr6[1].'<br/>';//1
echo $arr6[''].'<br/>';//a
echo $arr6[""].'<br/>';//a
$arr6[5] = 'e';
echo count($arr6).'<br/>';//3
echo $arr6[5].'<br/>';//3

3.下标为浮点数、浮点数小标会并列在整数下标内、并不影响下标数量
$arr6[5.1] = 'e.1';
echo count($arr6).'<br/>';//4
echo $arr6[5.1].'<br/>';//e.1
foreach($arr6 as $key=>$val){
	echo $key.'---'.$val.'<br/>';
};//1--1 //0--0 //---a //5---e.1
echo $arr6[5].'<br/>';//e.1
echo $arr6[5.1].'<br/>';//e.1
如果没有给数组指定key、则取当前最大的整数索引值、而新的键名讲是该值+1

4.键值若是字符串、需带引号、(新版本可能不能调用)
 若不带引号、有可能出现键值是常量
 const a = 'aa';
 $arr[a] = 1;
 echo $arr[a];//报错
字符串拆分转换数组 
$str = 'a b c d e f';
$arr8 = explode(' ',$str);
print_r($arr8);
//Array ( [0] => a [1] => b [2] => c [3] => d [4] => e [5] => f )
删除数组内容 unset($arr[num]) 
删除数组中的元素、unset($arr[num])
删除后索引值不会自动排队
数组运算符 
$arr1 +   $arr2  联合  $arr1 和 $arr2的联合、
 存在相同的键不会覆盖arr1的值
$arr1 ==  $arr2  相等  具有相同的键/值返回true
$arr1 === $arr2  全等  具有相同的键/值/顺序返回true
$arr1 !=  $arr2  不等  若不等于返回true
$arr1 <>  $arr2  不等  若不等于返回true
$arr1 !== $arr2  不全等  不全等于返回true

关闭notice级别的提示 
1.在变量前+@(非致命错误继续运行)

在php.ini文件中改动error_reporting
改为error_reporting = E_ALL  &  ~E_NOTICE
如果你不恩那个操作php.ini文件、你可以用

在你想进制notice错误提示的页面中加入下面的代码
/* Report all errors except E_NOTICE*/
error_reporting(E_ALL ^ E_NOTICE);
