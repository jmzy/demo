2.双击安装、在开始栏打开Node.js command prompt
3.cd C:\Program Files\nodejs

查看 npm -v 和 node -v 版本号
升级NPM ：npm install npm -g
升级NODE：npm install npm -g
安装 AngularJS 步骤 
安装Angular 必须和node不在同1个文件夹下
官方地址
cli.angular.io
                	
1.安装全局cli
npm install -g @angular/cli
出现问题重复、卸载掉旧版cli，下载最新版
npm uninstall -g angular-cli
npm cache clean
npm install -g angular-cli@latest

2.创建 angular 项目 (名称)
ng new my-dream-app

3.进入项目
cd my-dream-app

4.把服务启动
ng serve

安装TypeScript
npm install -g typescript typings
