Angular常用指令

ng-bind 和 ng-cloak 
写在 DOM 元素上、隐藏其加载的时候出现的{{}}(两个大括号)

事例
<div ng-app="app">
    <div ng-controller="firstCtrl">
        <h1 ng-bind="1+2"></h1>
        <h1 ng-cloak>{{1+2}}</h1>
    </div>
</div>
<script src="../../js/angular.min.js"></script>
<script>
    angular.module('app',[])
    .controller('firstCtrl',function($scope){
    })
</script>
ng-show & ng-hide 和 ng-if 
这3个命令作用都是判断其内容是否为 true 和 false
ng-show 和 ng-if 内容为 true 其内容显示、false 隐藏
ng-hide ture 隐藏、false 显示
★ng-if 与 ng-show ng-hide 区别在于false会让其 HTML 消失

事例
<div ng-app="appT">
    <div ng-controller="firstCtrl">
        <h1 ng-bind="1+2" ng-show="false"></h1><!-- HTML代码存在-->
        <h1 ng-cloak ng-hide="true">{{1+2}}</h1><!-- HTML代码存在-->
        <h1 ng-cloak ng-if="false">{{1+1}}</h1><!-- ngIf: false -->
    </div>
</div>
<script src="../../js/angular.min.js"></script>
<script>
    angular.module('appT',[])
    .controller('firstCtrl',function($scope){
    })
</script>
ng-src 和 ng-class 
引用图片的路径、在元素上编写、需要+{{}}

事例
<div ng-app="appThree">
    <div ng-controller="firstCtrl">
        <img src="../../images/css/hack.png" ng-src="{{a.img}}"
             ng-class="{'a':a.true}" />
        <!-- ★注意 仅显示Angular ng-src的图片-->
        <!--需引号指定class name ：true使用 false 不使用
            或｛'a':a.true｝
            ★若不输入有误则会显示原src的图片
        -->
    </div>
</div>
<script src="../../js/angular.min.js"></script>
<script>
    angular.module('appT',[])
    .controller('firstCtrl',function($scope){
        $scope.a ={
            img:'../../images/3.3/1.jpg',
            true:true
        }
    })
</script>
ng-checked 单选和复选框 
单选框事例

<div ng-app="appT">
    <div ng-controller="firstCtrl">
        <input type="radio" ng-checked="a.sex == 'm'" name="sex"/>男
        <input type="radio" ng-checked="a.sex == 'g'" name="sex"/>女
        <!-- ng-checked 值要注意是否是字符串 -->
    </div>
</div>
<script src="../../js/angular.min.js"></script>
<script>
    angular.module('appT',[])
    .controller('firstCtrl',function($scope){
        $scope.a ={
            sex:'g'//注意是字符串
        }
    })
</script>

JavaScript一种直译式脚本语言，是一种动态类型、弱类型、基于原型的语言，内置支持类型。它的解释器被称为JavaScript引擎，为浏览器的一部分，广泛用于客户端的脚本语言，最早是在HTML（标准通用标记语言下的一个应用）网页上使用，用来给HTML网页增加动态功能。 为了取得技术优势，微软推出了JScript，CEnvi推出ScriptEase，与JavaScript同样可在浏览器上运行。为了统一规格，因为JavaScript兼容于ECMA标准，因此也称为ECMAScript。 在1995年时，由Netscape公司的Brendan Eich，在网景导航者浏览器上首次设计实现而成。因为Netscape与Sun合作，Netscape管理层希望它外观看起来像Java，因此取名为JavaScript。但实际上它的语法风格与Self及Scheme较为接近。
