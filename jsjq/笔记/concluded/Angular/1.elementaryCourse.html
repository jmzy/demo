AngularJS 起步、代码简单分析

简单实例 
<div ng-app="myApp" ng-controller="myCtrl">
  <h1>{{msg}}</h1>
</div>

<script>
var app = angular.module('myApp', []);
app.controller('myCtrl', function($scope){
  $scope.msg = "hello AngularJS";
});
</script>
★关于ng-app 
ng-app 是声明angular app模块的是的开始
ng-app 的绑定会影响你的根在哪里
  可以绑在html body 或者 div...上 、绑定后、其根就在html...

★1个页面绑定两个ng-app 
★2个页面绑定同1个ng-app
关于Angular.module(参考于 破浪 博客) 原文出处 
module是angular代码的入口，首先需要声明module，然后才能定义
angular中的其他组件元素，如controller、service、filter、directive、
config代码块、run代码块等。
★关于.controller 
controller是一个javascript函数/类，用于操作作用域中，各个对象的初始状态
以及相应的行为

对元素使用 ng-controller 指令则该元素为了绑定的作用范围，

事例
<div ng-app="app">
  <h3 ng-controller="firstCtrl">
	{{1+2}}
  </h3>
</div>

<script src="../../js/angular.min.js"></script>
<script>
  angular.module('app',[])
  .controller('firstCtrl',function($scope){
  })
</script>

对<h3>元素使用 ng-controller 指令、控制器为firstCtrl、则该区域都为
firstCtrl的作用范围、因此输入angularJS命令{{ }} 就可以执行
关于$scope 和 {{ }} 取值表达式 出处 菜鸟教程 更详细 
$scope(作用域) 是应用在 HTML (视图) 和 JavaScript (控制器)之间的纽带。

如何使用 $scope 和 {{}} 简单事例
<div ng-app="myApp" ng-controller="myCtrl">
  <h1>{{msg}}</h1>· // {{ 表达式 }}
</div>

<script>
var app = angular.module('myApp', []);
app.controller('myCtrl', function($scope) {
  $scope.msg = "hello AngularJS";
});
</script>

AngularJS 表达式写在双大括号内：{{ 内容 }}。
非对象字符串要+双引号之类的
