VUE2 指令(一共就13个指令,非常简单)

v-text 和 v-html 
v-text
作用跟{{}}效果一样
<p v-text="msg"></p>
<p>{{msg}}</p>
data:{
    msg: 'test v-text',
}

v-html
尽量不要使用,原因https://cn.vuejs.org/v2/api/#v-html
<div v-html="html"></div>
data:{
    html: '<div style="margin:15px auto;width:100px;height:50px;background:#f60"></div>'
}
v-pre 
没搞懂
v-show 和 v-if 和 v-else 和 v-else-if 
跟vue1是一样的,
不符合条件:
v-show会保留元素,内联样式添加display:none
v-if不加载不符合条件的元素
v-for 
根据判断的条件编列数据,最好匹配:key使用
v-bind 
看官网吧
v-model 
demo

修饰符：

.lazy - 取代 input 监听 change 事件
.number - 输入字符串转为数字
.trim - 输入首尾空格过滤
v-on 
demo

修饰符：

.stop - 调用 event.stopPropagation()。
.prevent - 调用 event.preventDefault()。
.capture - 添加事件侦听器时使用 capture 模式。
.self - 只当事件是从侦听器绑定的元素本身触发时才触发回调。
.{keyCode | keyAlias} - 只当事件是从特定键触发时才触发回调。
.native - 监听组件根元素的原生事件。
.once - 只触发一次回调。
.left - (2.2.0) 只当点击鼠标左键时触发。
.right - (2.2.0) 只当点击鼠标右键时触发。
.middle - (2.2.0) 只当点击鼠标中键时触发。
.passive - (2.3.0) 以 { passive: true } 模式添加侦听器
v-cloak 
配合使用(css)
[v-cloak] {
  display: none;
}
作用在页面加载成功后再展示该元素
v-once 
只渲染元素和组件一次
